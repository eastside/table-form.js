table-form.js Demo

Simple form table

In this example, we have a simple HTML table that contains three input elements, which are all blank. Form tables with no initial values begin with only the first row visible.

Try adding some rows, entering random data, and deleting them. Notice how the table appears to act like it would in a common table editor like excel: Deleting a middle row causes the data from the next row to fill it, and a row is removed from the end of the list.

This JavaScript module follows the principle of progressive enhancement: if JavaScript was disabled, the form would still work just fine -- all fields would just be visible and could be filled out and submitted.
